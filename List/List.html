1. Singly Linked List

Concept

Each node contains data + next pointer (points to next node).

You can only traverse forward (no backward movement).

Last node’s next = None (or null).

Advantages

  • Simple implementation
  • Memory efficient (only one pointer per node)
  • Good for sequential access, stack/queue implementation

Disadvantages

  • No backward traversal
  • Deleting a node requires previous node’s reference (O(n) in worst case without prev pointer)

Basic Node Structure & Operations

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class SinglyLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return

        curr = self.head
        while curr.next:
            curr = curr.next
        curr.next = new_node

    def print_list(self):
        curr = self.head
        while curr:
            print(curr.data, end=" → ")
            curr = curr.next
        print("None")

# Example usage
ll = SinglyLinkedList()
ll.append(10)
ll.append(20)
ll.append(30)
ll.append(40)
ll.print_list()          # Output: 10 → 20 → 30 → 40 → None

2. Doubly Linked List

Concept

Each node has data + next pointer + prev pointer.

Can traverse forward and backward.

Advantages

  • Easy deletion (if you have the node, you can remove it in O(1) using prev & next)
  • Bidirectional traversal
  • Useful in browser history, LRU cache, etc.

Disadvantages

  • Uses more memory (two pointers per node)
  • More complex insertion/deletion logic

Basic Implementation (Python)

class DoublyNode:
    def __init__(self, data):
        self.data = data
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = DoublyNode(data)
        if not self.head:
            self.head = new_node
            return

        curr = self.head
        while curr.next:
            curr = curr.next

        curr.next = new_node
        new_node.prev = curr

    def print_forward(self):
        curr = self.head
        while curr:
            print(curr.data, end=" ↔ ")
            curr = curr.next
        print("None")

    def print_backward(self):
        if not self.head:
            return
        curr = self.head
        while curr.next:
            curr = curr.next
        # Now curr is tail
        while curr:
            print(curr.data, end=" ↔ ")
            curr = curr.prev
        print("None")

# Example
dll = DoublyLinkedList()
dll.append(1)
dll.append(2)
dll.append(3)
dll.print_forward()     # 1 ↔ 2 ↔ 3 ↔ None
dll.print_backward()    # 3 ↔ 2 ↔ 1 ↔ None

3. Fast-Slow Pointers (Tortoise & Hare Technique)

Concept

Two pointers moving at different speeds in the same linked list:

  • Slow (tortoise) → moves 1 step
  • Fast (hare) → moves 2 steps

Main Uses

  1. Find middle of linked list (very common in interviews)
  1. Detect cycle/loop in linked list → Floyd’s Cycle Detection Algorithm (O(n) time, O(1) space)
  1. Find starting point of cycle (after detection)

Example 1: Find Middle (odd & even length)

def find_middle(head):
    if not head:
        return None
    slow = head
    fast = head

    while fast and fast.next:     # fast can take 2 steps
        slow = slow.next
        fast = fast.next.next

    return slow.data if slow else None

# 1 → 2 → 3 → 4 → 5    → middle = 3
# 1 → 2 → 3 → 4        → middle = 3 (or 2 depending on definition)

Example 2: Detect Cycle (Floyd’s Algorithm)

def has_cycle(head):
    if not head or not head.next:
        return False

    slow = head
    fast = head

    while fast and fast.next:
        slow = slow.next              # +1
        fast = fast.next.next         # +2

        if slow == fast:
            return True               # cycle exists

    return False

How it works (when cycle exists)

Fast catches up to slow inside the loop (mathematically proven).

1. Cycle Detection (Floyd's Tortoise and Hare Algorithm)

Detects if a singly linked list has a cycle (loop) efficiently in O(n) time and O(1) space.

Idea — Use two pointers: slow (tortoise) moves 1 step, fast (hare) moves 2 steps. If they meet → cycle exists.

Python Implementation

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def hasCycle(head: ListNode) -> bool:
    if not head or not head.next:
        return False

    slow = head
    fast = head

    while fast and fast.next:
        slow = slow.next          # 1 step
        fast = fast.next.next     # 2 steps

        if slow == fast:
            return True           # Cycle detected

    return False

2. Reverse a Linked List

Reverse the direction of all pointers in a singly linked list.

Iterative Approach (most common & efficient — O(n) time, O(1) space)

Python Implementation

def reverseList(head: ListNode) -> ListNode:
    prev = None
    curr = head

    while curr:
        next_temp = curr.next    # save next
        curr.next = prev         # reverse pointer
        prev = curr              # move prev forward
        curr = next_temp         # move curr forward

    return prev                  # new head

3. Merge Two Sorted Linked Lists

Merge two sorted linked lists into one sorted linked list.

Python Implementation (iterative with dummy node)

def mergeTwoLists(list1: ListNode, list2: ListNode) -> ListNode:
    dummy = ListNode(0)
    current = dummy

    while list1 and list2:
        if list1.val <= list2.val:
            current.next = list1
            list1 = list1.next
        else:
            current.next = list2
            list2 = list2.next
        current = current.next

    # Attach remaining nodes
    current.next = list1 if list1 else list2

    return dummy.next

4. Find Intersection of Two Linked Lists (Y-shaped)

Find the starting node where two linked lists merge (if they intersect).

Best Method → Calculate lengths → align starting points → move together until match.

Python Implementation

def getIntersectionNode(headA: ListNode, headB: ListNode) -> ListNode:
    if not headA or not headB:
        return None

    # Find lengths
    lenA = lenB = 0
    pA, pB = headA, headB

    while pA:
        lenA += 1
        pA = pA.next
    while pB:
        lenB += 1
        pB = pB.next

    # Align pointers (move longer one forward)
    pA, pB = headA, headB
    if lenA > lenB:
        for _ in range(lenA - lenB):
            pA = pA.next
    else:
        for _ in range(lenB - lenA):
            pB = pB.next

    # Move together until intersection
    while pA != pB:
        pA = pA.next
        pB = pB.next

    return pA  # intersection node or None
