Trees and BST

Introduction to Trees and Binary Search Trees (BST)

Trees are hierarchical data structures consisting of nodes connected by edges, with a root at the top and no cycles. Each node can have child nodes, forming branches. A Binary Tree limits each node to at most two children (left and right). A Binary Search Tree (BST) is a special binary tree where for every node, all left subtree values are less than the node's value, and all right subtree values are greater, enabling efficient searching, insertion, and deletion (average O(log n) time).

Now, focusing on problems and implementations for each subtopic. I'll provide example problems, followed by various implementations (e.g., recursive, iterative) in Python, using a basic TreeNode class for consistency.

class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

Tree Traversals

Traversals visit all nodes in a tree in specific orders. Common problems include printing node values in a given order, finding paths, or validating structures.

Inorder Traversal (Left-Root-Right)

Problem: Given a binary tree, return the inorder traversal of its nodes' values. Example: For tree [1, None, 2, 3], output [1, 3, 2].

  • Recursive Implementation:
    def inorderTraversal(root):
    def helper(node, result):
        if node:
            helper(node.left, result)
            result.append(node.val)
            helper(node.right, result)
    res = []
    helper(root, res)
    return res
  • Iterative Implementation (Using Stack):
    def inorderTraversal(root):
    res, stack = [], []
    curr = root
    while curr or stack:
        while curr:
            stack.append(curr)
            curr = curr.left
        curr = stack.pop()
        res.append(curr.val)
        curr = curr.right
    return res
  • Morris Traversal (Threaded, O(1) Space):
    def inorderTraversal(root):
    res = []
    curr = root
    while curr:
        if not curr.left:
            res.append(curr.val)
            curr = curr.right
        else:
            pred = curr.left
            while pred.right and pred.right != curr:
                pred = pred.right
            if not pred.right:
                pred.right = curr
                curr = curr.left
            else:
                pred.right = None
                res.append(curr.val)
                curr = curr.right
    return res

Preorder Traversal (Root-Left-Right)

Problem: Given a binary tree, return the preorder traversal. Example: For tree [1, None, 2, 3], output [1, 2, 3].

  • Recursive Implementation:
    def preorderTraversal(root):
    def helper(node, result):
        if node:
            result.append(node.val)
            helper(node.left, result)
            helper(node.right, result)
    res = []
    helper(root, res)
    return res
  • Iterative Implementation (Using Stack):
    def preorderTraversal(root):
    if not root:
        return []
    res, stack = [], [root]
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.right:
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    return res
  • Morris Traversal (Threaded, O(1) Space):
    def preorderTraversal(root):
    res = []
    curr = root
    while curr:
        if not curr.left:
            res.append(curr.val)
            curr = curr.right
        else:
            pred = curr.left
            while pred.right and pred.right != curr:
                pred = pred.right
            if not pred.right:
                res.append(curr.val)
                pred.right = curr
                curr = curr.left
            else:
                pred.right = None
                curr = curr.right
    return res

Postorder Traversal (Left-Right-Root)

Problem: Given a binary tree, return the postorder traversal. Example: For tree [1, None, 2, 3], output [3, 2, 1].

  • Recursive Implementation:
    def postorderTraversal(root):
    def helper(node, result):
        if node:
            helper(node.left, result)
            helper(node.right, result)
            result.append(node.val)
    res = []
    helper(root, res)
    return res
  • Iterative Implementation (Using Two Stacks):
    def postorderTraversal(root):
    if not root:
        return []
    res, stack1, stack2 = [], [root], []
    while stack1:
        node = stack1.pop()
        stack2.append(node)
        if node.left:
            stack1.append(node.left)
        if node.right:
            stack1.append(node.right)
    while stack2:
        res.append(stack2.pop().val)
    return res
  • Iterative Implementation (Using One Stack with Flags):
    def postorderTraversal(root):
    res, stack = [], []
    node, last_visited = root, None
    while stack or node:
        if node:
            stack.append(node)
            node = node.left
        else:
            peek = stack[-1]
            if peek.right and last_visited != peek.right:
                node = peek.right
            else:
                res.append(peek.val)
                last_visited = stack.pop()
    return res
  • Morris Traversal (Threaded, O(1) Space, Reversed Preorder Variant):
    def postorderTraversal(root):
    def reverse(root):
        prev, curr = None, root
        while curr:
            next = curr.right
            curr.right = prev
            prev = curr
            curr = next
        return prev

    res = []
    dummy = TreeNode(0, right=root)
    curr = dummy
    while curr:
        if not curr.right:
            curr = curr.left
        else:
            pred = curr.right
            while pred.left and pred.left != curr:
                pred = pred.left
            if not pred.left:
                pred.left = curr
                curr = curr.right
            else:
                pred.left = None
                # Add reversed right subtree to res
                rev_head = reverse(curr.right)
                temp = rev_head
                while temp:
                    res.append(temp.val)
                    temp = temp.right
                reverse(rev_head)  # Restore
                curr = curr.left
    return res[::-1]  # Reverse res for postorder

Level-Order Traversal (BFS, Level by Level)

Problem: Given a binary tree, return the level-order traversal as a list of lists. Example: For tree [3,9,20,None,None,15,7], output [[3],[9,20],[15,7]].

  • Iterative Implementation (Using Queue):
    from collections import deque

def levelOrder(root):
    if not root:
        return []
    res = []
    queue = deque([root])
    while queue:
        level_size = len(queue)
        level = []
        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        res.append(level)
    return res
  • Recursive Implementation (Depth Tracking):
    def levelOrder(root):
    def helper(node, level, res):
        if not node:
            return
        if len(res) == level:
            res.append([])
        res[level].append(node.val)
        helper(node.left, level + 1, res)
        helper(node.right, level + 1, res)

    res = []
    helper(root, 0, res)
    return res

Tree Construction

Tree construction problems involve building a tree from given data like traversals or arrays. Common scenarios: Construct from inorder/preorder, or from level-order.

Construct Binary Tree from Inorder and Preorder Traversals

Problem: Given inorder = [9,3,15,20,7] and preorder = [3,9,20,15,7], build the tree.

  • Recursive Implementation:
    def buildTree(preorder, inorder):
    if not preorder or not inorder:
        return None
    root = TreeNode(preorder[0])
    mid = inorder.index(preorder[0])
    root.left = buildTree(preorder[1:mid+1], inorder[:mid])
    root.right = buildTree(preorder[mid+1:], inorder[mid+1:])
    return root
  • Iterative Implementation (Using Stack):
    def buildTree(preorder, inorder):
    if not preorder:
        return None
    root = TreeNode(preorder[0])
    stack = [root]
    i, j = 1, 0
    while i < len(preorder):
        node = TreeNode(preorder[i])
        if inorder[j] == stack[-1].val:
            while stack and inorder[j] == stack[-1].val:
                last = stack.pop()
                j += 1
            last.right = node
        else:
            stack[-1].left = node
        stack.append(node)
        i += 1
    return root

Construct Binary Tree from Inorder and Postorder Traversals

Problem: Given inorder = [9,3,15,20,7] and postorder = [9,15,7,20,3], build the tree.

  • Recursive Implementation:
    def buildTree(inorder, postorder):
    if not inorder or not postorder:
        return None
    root = TreeNode(postorder[-1])
    mid = inorder.index(postorder[-1])
    root.left = buildTree(inorder[:mid], postorder[:mid])
    root.right = buildTree(inorder[mid+1:], postorder[mid:-1])
    return root
  • Iterative Implementation (Using Stack):
    def buildTree(inorder, postorder):
    if not postorder:
        return None
    root = TreeNode(postorder[-1])
    stack = [root]
    i = len(postorder) - 2
    inorder_idx = len(inorder) - 1
    while i >= 0:
        node = TreeNode(postorder[i])
        if stack[-1].val != inorder[inorder_idx]:
            stack[-1].right = node
        else:
            while stack and stack[-1].val == inorder[inorder_idx]:
                last = stack.pop()
                inorder_idx -= 1
            last.left = node
        stack.append(node)
        i -= 1
    return root

Construct BST from Preorder Traversal

Problem: Given preorder = [8,5,1,7,10,12], build the BST.

  • Recursive Implementation:
    def bstFromPreorder(preorder):
    def helper(values, low, high):
        if low > high:
            return None
        root = TreeNode(values[low])
        mid = low + 1
        while mid <= high and values[mid] < root.val:
            mid += 1
        root.left = helper(values, low + 1, mid - 1)
        root.right = helper(values, mid, high)
        return root
    return helper(preorder, 0, len(preorder) - 1)
  • Iterative Implementation (Using Stack):
    def bstFromPreorder(preorder):
    if not preorder:
        return None
    root = TreeNode(preorder[0])
    stack = [root]
    for val in preorder[1:]:
        node = TreeNode(val)
        if val < stack[-1].val:
            stack[-1].left = node
        else:
            while stack and val > stack[-1].val:
                last = stack.pop()
            last.right = node
        stack.append(node)
    return root

BST Operations

BST operations maintain the BST property. Common problems: Insert a key, delete a key, search for a key, find min/max, successor/predecessor.

Search in BST

Problem: Given root and key, return the node with that key or None.

  • Recursive Implementation:
    def searchBST(root, val):
    if not root or root.val == val:
        return root
    return searchBST(root.left, val) if val < root.val else searchBST(root.right, val)
  • Iterative Implementation:
    def searchBST(root, val):
    while root:
        if root.val == val:
            return root
        root = root.left if val < root.val else root.right
    return None

Insert into BST

Problem: Insert a new key into the BST while maintaining order.

  • Recursive Implementation:
    def insertIntoBST(root, val):
    if not root:
        return TreeNode(val)
    if val < root.val:
        root.left = insertIntoBST(root.left, val)
    else:
        root.right = insertIntoBST(root.right, val)
    return root
  • Iterative Implementation:
    def insertIntoBST(root, val):
    if not root:
        return TreeNode(val)
    curr = root
    while True:
        if val < curr.val:
            if not curr.left:
                curr.left = TreeNode(val)
                return root
            curr = curr.left
        else:
            if not curr.right:
                curr.right = TreeNode(val)
                return root
            curr = curr.right

Delete from BST

Problem: Delete a key from the BST, handling cases: no children, one child, two children (replace with inorder successor).

  • Recursive Implementation:
    def deleteNode(root, key):
    if not root:
        return None
    if key < root.val:
        root.left = deleteNode(root.left, key)
    elif key > root.val:
        root.right = deleteNode(root.right, key)
    else:
        if not root.left:
            return root.right
        if not root.right:
            return root.left
        # Find inorder successor
        temp = root.right
        while temp.left:
            temp = temp.left
        root.val = temp.val
        root.right = deleteNode(root.right, temp.val)
    return root
  • Iterative Implementation:
    def deleteNode(root, key):
    if not root:
        return None
    parent, curr = None, root
    while curr and curr.val != key:
        parent = curr
        curr = curr.left if key < curr.val else curr.right
    if not curr:
        return root
    # Node with two children
    if curr.left and curr.right:
        succ_parent, succ = curr, curr.right
        while succ.left:
            succ_parent = succ
            succ = succ.left
        curr.val = succ.val
        if succ_parent == curr:
            succ_parent.right = succ.right
        else:
            succ_parent.left = succ.right
    # Node with one or no child
    else:
        child = curr.left or curr.right
        if not parent:
            return child
        if curr == parent.left:
            parent.left = child
        else:
            parent.right = child
    return root

Find Minimum/Maximum in BST

Problem: Find the smallest/largest value.

  • Recursive Implementation (Min):
    def findMin(root):
    if not root:
        return None
    return findMin(root.left) or root.val
  • Iterative Implementation (Min):
    def findMin(root):
    if not root:
        return None
    while root.left:
        root = root.left
    return root.val
  • (Max implementations are symmetric, swapping left/right.)

Inorder Successor/Predecessor in BST

Problem: Find the next/previous node in inorder traversal.

  • Iterative Implementation (Successor):
    def inorderSuccessor(root, node):
    succ = None
    while root:
        if node.val < root.val:
            succ = root
            root = root.left
        else:
            root = root.right
    return succ
  • Iterative with Stack (For Predecessor, Symmetric):
    def inorderPredecessor(root, node):
    pred = None
    while root:
        if node.val > root.val:
            pred = root
            root = root.right
        else:
            root = root.left
    return pred

class TreeNode:

        self.right = right

1. Lowest Common Ancestor (LCA) of two nodes in a Binary Tree

Problem: Given root and two nodes p and q, find their lowest common ancestor (the deepest node that has both p and q as descendants).

Approach A – Recursive (most common & clean)

def lowestCommonAncestor(root: TreeNode, p: TreeNode, q: TreeNode) -> TreeNode:
    if not root or root == p or root == q:
        return root

    left = lowestCommonAncestor(root.left, p, q)
    right = lowestCommonAncestor(root.right, p, q)

    # If both sides found something → current node is LCA
    if left and right:
        return root

    # Otherwise return the non-None side (or None)
    return left if left else right

Approach B – Recursive with parent map (works for any tree, not just binary)

(Usually used when nodes don’t have parent pointers, but we build one)

def lowestCommonAncestor(root, p, q):
    parent = {root: None}

    def dfs(node):
        if node:
            if node.left:
                parent[node.left] = node
            if node.right:
                parent[node.right] = node
            dfs(node.left)
            dfs(node.right)

    dfs(root)

    # Get ancestors of p
    ancestors = set()
    curr = p
    while curr:
        ancestors.add(curr)
        curr = parent[curr]

    # Find first ancestor of q that is in p's ancestors
    curr = q
    while curr not in ancestors:
        curr = parent[curr]

    return curr

Approach C – For BST only (faster – O(h) without visiting whole tree)

def lowestCommonAncestorBST(root: TreeNode, p: TreeNode, q: TreeNode) -> TreeNode:
    if not root:
        return None

    # Make sure p.val <= q.val
    if p.val > q.val:
        p, q = q, p

    while root:
        if root.val < p.val:
            root = root.right
        elif root.val > q.val:
            root = root.left
        else:
            # root.val is between p and q (or equal to one of them)
            return root
    return None

2. Diameter of a Binary Tree

Diameter = length of the longest path between any two nodes in a tree (path may or may not pass through root).

Length = number of edges (not nodes).

Most Popular Solution – Post-order style (single pass)

class Solution:
    def diameterOfBinaryTree(self, root: TreeNode) -> int:
        self.diam = 0

        def height(node):
            if not node:
                return 0

            left_h = height(node.left)
            right_h = height(node.right)

            # Diameter through current node
            self.diam = max(self.diam, left_h + right_h)

            # Return height of this subtree
            return 1 + max(left_h, right_h)

        height(root)
        return self.diam

Alternative – Return both height and diameter (cleaner in some languages)

def diameterOfBinaryTree(root):
    def dfs(node):
        if not node:
            return 0, 0   # height, diameter

        left_h, left_d = dfs(node.left)
        right_h, right_d = dfs(node.right)

        height_here = 1 + max(left_h, right_h)
        diam_here = max(
            left_d,
            right_d,
            left_h + right_h   # path through current node
        )

        return height_here, diam_here

    _, diameter = dfs(root)
    return diameter

3. Check if Binary Tree is Height-Balanced

Definition: A binary tree is height-balanced if:

  • The left and right subtrees of every node differ in height by at most 1, and
  • Both subtrees are also height-balanced.

Approach A – Top-down (naive, O(n²) worst case)

def isBalanced(root: TreeNode) -> bool:
    def height(node):
        if not node:
            return 0
        return 1 + max(height(node.left), height(node.right))

    if not root:
        return True

    left_h = height(root.left)
    right_h = height(root.right)

    if abs(left_h - right_h) > 1:
        return False

    return isBalanced(root.left) and isBalanced(root.right)

Approach B – Bottom-up (best – O(n), most asked in interviews)

class Solution:
    def isBalanced(self, root: TreeNode) -> bool:

        def check(node):
            if not node:
                return 0   # height

            left = check(node.left)
            right = check(node.right)

            # -1 means unbalanced subtree found somewhere below
            if left == -1 or right == -1:
                return -1

            if abs(left - right) > 1:
                return -1

            return 1 + max(left, right)

        return check(root) != -1

One-liner style (Pythonic, still O(n))

def isBalanced(root):
    def dfs(node):
        if not node:
            return 0, True

        lh, lb = dfs(node.left)
        rh, rb = dfs(node.right)

        balanced = lb and rb and abs(lh - rh) <= 1
        return 1 + max(lh, rh), balanced

    return dfs(root)[1]
