Arrays & Strings

Array Manipulation

Basic but powerful: accessing, updating, inserting, deleting elements.

Time complexities matter a lot:

  • Access by index → O(1)
  • Insert/delete at beginning/end → O(n) (shifting elements)
  • Insert/delete in middle → O(n)

In Python, lists are dynamic arrays → amortized O(1) append, but insert(0) is O(n).

Common tasks: reverse, find max/min, sort (but prefer built-in when allowed), count frequencies (use dict or Counter).

Two Pointers

Very efficient technique (usually O(n) time) for sorted arrays or when we can maintain order.

Two pointers typically:

  • Start from both ends (left = 0, right = n-1) → good for pair sum, container with most water
  • Fast & slow pointers → cycle detection, middle of list
  • Same direction → sliding window style

Classic example: Find two numbers in sorted array that sum to target.

def twoSumSorted(nums, target):
    left, right = 0, len(nums) - 1
    while left < right:
        curr_sum = nums[left] + nums[right]
        if curr_sum == target:
            return [left, right]
        elif curr_sum < target:
            left += 1
        else:
            right -= 1
    return []

Sliding Window

Maintains a "window" of elements that satisfy some condition.

Two main types:

  • Fixed size (e.g. max sum of k consecutive elements)
  • Variable size (e.g. longest substring without repeating chars)

Key: Use two pointers (left & right) and expand/shrink window.

Fixed size example: Max sum subarray of size k

def maxSumSubarray(arr, k):
    if len(arr) < k:
        return 0

    # Initial window sum
    window_sum = sum(arr[:k])
    max_sum = window_sum

    # Slide the window
    for i in range(k, len(arr)):
        window_sum = window_sum - arr[i - k] + arr[i]
        max_sum = max(max_sum, window_sum)

    return max_sum

Prefix Sums

Precompute cumulative sums → allows O(1) range sum queries after O(n) preprocessing.

def build_prefix_sum(arr):
    prefix = [0] * (len(arr) + 1)
    for i in range(1, len(prefix)):
        prefix[i] = prefix[i-1] + arr[i-1]
    return prefix

# Range sum from l to r (inclusive)
def range_sum(prefix, l, r):
    return prefix[r+1] - prefix[l]

Example usage:

arr = [3, 1, 4, 1, 5, 9]
prefix = build_prefix_sum(arr)  # [0, 3, 4, 8, 9, 14, 23]
print(range_sum(prefix, 2, 5))  # 4+1+5+9 = 19

Kadane’s Algorithm

Finds maximum subarray sum in O(n) time — very famous problem.

Idea: At each position, decide whether to start a new subarray or extend the previous one.

def kadane(arr):
    if not arr:
        return 0

    max_ending_here = max_so_far = arr[0]

    for num in arr[1:]:
        max_ending_here = max(num, max_ending_here + num)
        max_so_far = max(max_so_far, max_ending_here)

    return max_so_far

Handles all-negative case too (just return max element).

# All negative example
print(kadane([-2, -3, -1, -5]))  # → -1

Array Rotation

Rotate array left/right by k positions.

Best way (O(n) time, O(1) space): Reversal algorithm

To rotate left by k:

  1. Reverse first k elements
  1. Reverse remaining n-k elements
  1. Reverse entire array
def rotate_left(arr, k):
    n = len(arr)
    k = k % n
    if k == 0:
        return

    # Step 1
    arr[:k] = reversed(arr[:k])
    # Step 2
    arr[k:] = reversed(arr[k:])
    # Step 3
    arr[:] = reversed(arr)

Dutch National Flag Problem

Sort array containing only 0s, 1s, 2s in one pass (O(n) time, O(1) space).

Use three pointers: low, mid, high

  • low → boundary for 0s (everything before low is 0)
  • mid → current element being processed
  • high → boundary for 2s (everything after high is 2)
def dutch_national_flag(arr):
    low = mid = 0
    high = len(arr) - 1

    while mid <= high:
        if arr[mid] == 0:
            arr[low], arr[mid] = arr[mid], arr[low]
            low += 1
            mid += 1
        elif arr[mid] == 1:
            mid += 1
        else:  # arr[mid] == 2
            arr[mid], arr[high] = arr[high], arr[mid]
            high -= 1

    return arr
