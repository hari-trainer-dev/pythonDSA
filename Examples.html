2 Pointer

  1. Check if an array is sorted

Very basic use — just walk through the array once comparing each element with the next one.

def is_sorted(arr):
    for i in range(1, len(arr)):
        if arr[i] < arr[i-1]:
            return False
    return True

print(is_sorted([1, 2, 3, 4, 5]))   # True
print(is_sorted([1, 3, 2, 4, 5]))   # False

(You can also write it with two pointers called left/right, but the simple loop above is usually clearer.)

  1. Find if two numbers add up to a target (sorted array)

Classic problem. Start one pointer at the beginning, one at the end, and move them based on the current sum.

def has_pair_with_sum(arr, target):
    left = 0
    right = len(arr) - 1

    while left < right:
        current_sum = arr[left] + arr[right]

        if current_sum == target:
            return True
        elif current_sum < target:
            left += 1
        else:
            right -= 1

    return False

print(has_pair_with_sum([1, 2, 3, 4, 6], 6))    # True (2+4)
  1. Remove duplicates from a sorted array (in-place)

Use a "slow" pointer to track where the next unique element should go.

def remove_duplicates(arr):
    if not arr:
        return []

    write = 0
    for read in range(1, len(arr)):
        if arr[read] != arr[write]:
            write += 1
            arr[write] = arr[read]

    return arr[:write + 1]

print(remove_duplicates([1, 1, 2, 2, 3, 3, 4]))  # [1, 2, 3, 4]
  1. Check if a string is a palindrome

Compare characters moving inward from both ends.

def is_palindrome(s):
    left = 0
    right = len(s) - 1

    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1

    return True

print(is_palindrome("madam"))     # True
print(is_palindrome("racecar"))   # True
print(is_palindrome("hello"))     # False

2D array search

  1. Basic linear search
    Works on any matrix, sorted or not. Just check every element.
def search_matrix(matrix, target):

    for row in matrix:
        for num in row:
            if num == target:
                return True
    return False

# Example
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(search_matrix(matrix, 5))  # True

Time complexity: O(rows × cols) — fine for small matrices, slow for large ones.

  1. Binary search by treating the matrix as a flattened 1D array
    Best when the matrix is fully sorted row-wise and the first element of each row is larger than the last element of the previous row (like a sorted list wrapped into rows).

Example valid matrix:

1  3  5
7  9 11
13 15 17
def search_matrix(matrix, target):

    rows = len(matrix)
    cols = len(matrix[0])

    left = 0
    right = rows * cols - 1

    while left <= right:
        mid = (left + right) // 2
        row = mid // cols
        col = mid % cols
        value = matrix[row][col]

        if value == target:
            return True
        elif value < target:
            left = mid + 1
        else:
            right = mid - 1

    return False

# Example
matrix = [
    [1, 3, 5],
    [7, 9, 11],
    [13, 15, 17]
]
print(search_matrix(matrix, 9))   # True

Time complexity: O(log(rows × cols)) — very efficient.

  1. Staircase / corner search (most popular in interviews)
    Works when each row is sorted and each column is sorted (but rows don't need to connect like in method 2).

Example valid matrix:

1  4  7
2  5  8
3  6  9

Start from the top-right corner.

  • If current value == target → found
  • If current value > target → move left (values get smaller)
  • If current value < target → move down (values get larger)
def search_matrix(matrix, target):
    

    rows = len(matrix)
    cols = len(matrix[0])

    row = 0
    col = cols - 1

    while row < rows and col >= 0:
        if matrix[row][col] == target:
            return True
        elif matrix[row][col] > target:
            col -= 1
        else:
            row += 1

    return False

# Example
matrix = [
    [1, 4, 7],
    [2, 5, 8],
    [3, 6, 9]
]
print(search_matrix(matrix, 5))   # True

Time complexity: O(rows + cols) — excellent, and the logic is very easy to explain in interviews.
